Final Project

Final Project

This year, everyone will work on the final project collaboratively on GitHub. We have invited all students to https://github.com/CBB752Spring2016/CBB752Spring2016.github.io. If you did not receive the invitation, please contact TAs immediately at cbb752 (at) gersteinlab.org.

We have posted a draft proposal for the final project as a markdown file (index.md) and it is also viewable from website http://CBB752Spring2016.github.io.

This is a draft proposal and we need everyone's participation. Feel free to make edits on topics and form your team.
